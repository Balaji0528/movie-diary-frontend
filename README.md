# MovieDiary Frontend

MovieDiary is a movie tracking web application inspired by Letterboxd. Users can discover movies, build a watchlist, maintain a personal movie diary, and review films they have watched.

## Features

* User Authentication (JWT)
* Browse Popular Movies
* Browse Trending Movies
* Movie Search with Suggestions
* Movie Details Page
* Add Movies to Watchlist
* Personal Movie Diary
* Movie Ratings and Reviews
* User Profile
* Loading Skeletons
* Responsive Design

## Tech Stack

* React
* Vite
* Tailwind CSS
* React Router DOM
* Axios
* React Hot Toast

## Installation

```bash
npm install
npm run dev
```

## Backend Repository

Connects to the MovieDiary Spring Boot backend API.
